## Что чинил по ходу

1. Unsplash по photo-ID отдавал случайные фото (офис, дельфин, гидросамолёт) → фотографии вырезаны кропами из референс-скриншота (1400px — для превью достаточно).
2. Кроп секции Itinerary захватывал текст hero из скриншота → перекроп.
3. Tailwind CDN (warning + медленно) → скомпилированный минифицированный styles.css (tailwindcss 3.4.17).
4. Контраст BOOK NOW (3.0:1) → bg #c93636 (5.2:1).
5. LCP 7.9s → hero пережат 1920→1600 q72 (793→348 KB) + preload/fetchpriority, Perf 70→78.
6. GitHub Pages деплой упал («Deployment failed, try again later», rerun завис в очереди) → переключился на Vercel CLI (аккаунт oxylok уже залогинен).
7. Повторный билд Tailwind из папки site/ обнулил контент-скан → styles.css без утилит, вёрстка развалилась. Пересобран из корня; добавлен build.sh как защита.
